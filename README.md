# barbershopify

Turn any song — or any poem — into a TTBB barbershop quartet arrangement, rendered as
interactive sheet music you can play back, edit, and export.

> **Status: under construction.** This README will grow into a full setup guide
> (prerequisites, one-command start, usage walkthrough) as the app takes shape.
> For the full project specification, see [SPEC.md](SPEC.md).

## What it will do

- **Upload a song** (.mp3, .m4a, .wav) → the app analyzes its melody, chords, and lyrics,
  then generates a four-part barbershop arrangement (Tenor / Lead / Baritone / Bass).
- **Paste your own lyrics** under an analyzed melody, or skip audio entirely and have an
  original arrangement composed from a poem's rhythm and mood.
- **See real sheet music** in the browser, with synchronized four-part playback (mute or
  solo any voice to learn your part), note and lyric editing, and MusicXML/MIDI export.
- A **"spice" dial (1–5)** controls how adventurous the arrangement gets — from faithful
  and singable to contest-style showpiece.
